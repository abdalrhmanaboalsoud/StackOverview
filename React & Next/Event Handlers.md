| **Event Handler**  | **Description**                                    | **Example**                                                   | **Example with `useState`**                                   |
|--------------------|----------------------------------------------------|---------------------------------------------------------------|---------------------------------------------------------------|
| `onClick`          | Triggered when an element is clicked. Used to handle user actions like button presses. | `<button onClick={handleClick}>Click Me!</button>`            | `const [count, setCount] = useState(0);`<br>`<button onClick={() => setCount(count + 1)}>Click Me!</button>` |
| `onChange`         | Triggered when the value of an input, select, or textarea changes. Commonly used to capture form input. | `<input type="text" onChange={handleChange} />`               | `const [text, setText] = useState('');`<br>`<input type="text" onChange={(e) => setText(e.target.value)} />` |
| `onSubmit`         | Triggered when a form is submitted. Prevents default form submission and can be used to handle form data via JavaScript. | `<form onSubmit={handleSubmit}>...</form>`                    | `const [formData, setFormData] = useState({});`<br>`<form onSubmit={(e) => { e.preventDefault(); setFormData({/* your form data */}); }}>...</form>` |
| `onFocus`          | Triggered when an element, typically an input field, gains focus. Used to highlight or validate fields as they are selected. | `<input type="text" onFocus={handleFocus} />`                 | `const [focused, setFocused] = useState(false);`<br>`<input type="text" onFocus={() => setFocused(true)} />` |
| `onBlur`           | Triggered when an element loses focus. Often used to validate input after the user leaves the field. | `<input type="text" onBlur={handleBlur} />`                   | `const [focused, setFocused] = useState(true);`<br>`<input type="text" onBlur={() => setFocused(false)} />` |
| `onKeyPress`       | Triggered when a key is pressed down. Captures the character typed and is useful for detecting specific key inputs. | `<input type="text" onKeyPress={handleKeyPress} />`           | `const [keyPressed, setKeyPressed] = useState('');`<br>`<input type="text" onKeyPress={(e) => setKeyPressed(e.key)} />` |
| `onKeyDown`        | Triggered when a key is pressed. This event occurs before the character is added to the input, making it ideal for handling keyboard shortcuts. | `<input type="text" onKeyDown={handleKeyDown} />`             | `const [keyDown, setKeyDown] = useState('');`<br>`<input type="text" onKeyDown={(e) => setKeyDown(e.key)} />` |
| `onKeyUp`          | Triggered when a key is released. Useful for capturing the final state of the input after typing. | `<input type="text" onKeyUp={handleKeyUp} />`                 | `const [keyUp, setKeyUp] = useState('');`<br>`<input type="text" onKeyUp={(e) => setKeyUp(e.key)} />` |
| `onMouseEnter`     | Triggered when the mouse pointer enters the bounds of an element. Often used for hover effects, such as highlighting. | `<div onMouseEnter={handleMouseEnter}>Hover over me!</div>`   | `const [hovered, setHovered] = useState(false);`<br>`<div onMouseEnter={() => setHovered(true)}>Hover over me!</div>` |
| `onMouseLeave`     | Triggered when the mouse pointer leaves the bounds of an element. Used to remove hover effects. | `<div onMouseLeave={handleMouseLeave}>Hover over me!</div>`   | `const [hovered, setHovered] = useState(true);`<br>`<div onMouseLeave={() => setHovered(false)}>Hover over me!</div>` |
| `onMouseOver`      | Triggered repeatedly when the mouse is over an element or any of its children. Similar to `onMouseEnter`, but with continuous firing as the mouse moves over child elements. | `<div onMouseOver={handleMouseOver}>Hover over me!</div>`     | `const [hovered, setHovered] = useState(false);`<br>`<div onMouseOver={() => setHovered(true)}>Hover over me!</div>` |
| `onMouseOut`       | Triggered when the mouse pointer leaves the element or any of its children. Similar to `onMouseLeave`, but with continuous firing as the mouse moves out of child elements. | `<div onMouseOut={handleMouseOut}>Hover over me!</div>`       | `const [hovered, setHovered] = useState(true);`<br>`<div onMouseOut={() => setHovered(false)}>Hover over me!</div>` |
| `onDoubleClick`    | Triggered when an element is double-clicked. Can be used to trigger an alternative action compared to a single click. | `<button onDoubleClick={handleDoubleClick}>Double Click</button>` | `const [doubleClicked, setDoubleClicked] = useState(false);`<br>`<button onDoubleClick={() => setDoubleClicked(true)}>Double Click</button>` |
| `onContextMenu`    | Triggered when the context menu (usually right-click) is requested. Can be used to display custom context menus. | `<div onContextMenu={handleContextMenu}>Right-click me!</div>`| `const [contextClicked, setContextClicked] = useState(false);`<br>`<div onContextMenu={() => setContextClicked(true)}>Right-click me!</div>` |
| `onInput`          | Triggered immediately when the value of an input element changes. Useful for real-time form validation. | `<input type="text" onInput={handleInput} />`                 | `const [input, setInput] = useState('');`<br>`<input type="text" onInput={(e) => setInput(e.target.value)} />` |
| `onDragStart`      | Triggered when a drag operation starts. Used in conjunction with drag-and-drop operations. | `<div onDragStart={handleDragStart}>Drag me!</div>`           | `const [dragging, setDragging] = useState(false);`<br>`<div onDragStart={() => setDragging(true)}>Drag me!</div>` |
| `onDragOver`       | Triggered when a dragged element is over a drop target. Allows you to control what happens when an item is dragged over a specific area. | `<div onDragOver={handleDragOver}>Drag over me!</div>`        | `const [dragOver, setDragOver] = useState(false);`<br>`<div onDragOver={() => setDragOver(true)}>Drag over me!</div>` |
| `onDrop`           | Triggered when a dragged element is dropped on a target. Used to handle the drop operation in a drag-and-drop interface. | `<div onDrop={handleDrop}>Drop here!</div>`                   | `const [dropped, setDropped] = useState(false);`<br>`<div onDrop={() => setDropped(true)}>Drop here!</div>` |
| `onScroll`         | Triggered when an element is scrolled. Useful for loading more content as the user scrolls, or for animations tied to scroll position. | `<div onScroll={handleScroll}>Scroll me!</div>`               | `const [scrolled, setScrolled] = useState(false);`<br>`<div onScroll={() => setScrolled(true)}>Scroll me!</div>` |
| `onResize`         | Triggered when the window or a specified element is resized. Typically used to adjust layout or content based on the new size. | `window.addEventListener('resize', handleResize);`            | `const [windowSize, setWindowSize] = useState({ width: window.innerWidth, height: window.innerHeight });`<br>`window.addEventListener('resize', () => setWindowSize({ width: window.innerWidth, height: window.innerHeight }));` |
